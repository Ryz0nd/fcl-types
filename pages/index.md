# fcl-types

The repository for type definitions of [`@onflow/fcl`](https://github.com/onflow/fcl-js) library.

## How to Install?

Add the following to your project's `package.json`.

```json
{
  "devDependencies": {
    "@types/onflow__fcl": "alpha-dao/fcl-types#v0.0.2",
  }
}
```
